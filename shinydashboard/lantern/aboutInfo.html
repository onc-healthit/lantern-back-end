How to use the website

1. Dashboard Tab

Displays a high-level summary of the FHIR endpoints data collected by Lantern. It shows the FHIR endpoints count per developer along with HTTP responses of FHIR endpoints per developer.

2. Endpoints Tab

Displays detailed information for each FHIR endpoint. Clicking on a FHIR endpoint URL will display a popup window containing all the information collected for that endpoint by Lantern. The data displayed in all tabs can be viewed for each FHIR endpoint in the popup window that appears for that endpoint. Additionally, you can download the summary report of all FHIR endpoints as a CSV file from this tab.

3. Organizations Tab

Here you can see HTI-1 rule-compliant information of all healthcare providers associated with the FHIR endpoints collected by Lantern. Clicking on the endpoint opens the popup window containing endpoint details. Additionally, you can download the summary report of all healthcare providers as a CSV file from this tab.

4. Resources Tab

Displays the distribution of the FHIR resources and their supported operations per developer. This tab has a “How to use this page” button which gives details on how to use the filters present on the page.

5. Implementation Guides Tab

Displays the distribution of the different implementation guides supported by the different FHIR endpoints.

6. Cap Stat Fields Tab

Displays the distribution of the fields present in the Capability Statement categorized in three different types: required fields, optional fields and extensions.

7. Cap Stat Values Tab

Displays the distribution of the values populated in certain fields of the Capability Statement. The list of fields you can view the values for is shown in the “Field” dropdown of this tab.

8. Cap Stat Profiles Tab

Displays the FHIR profiles supported by each resource for all FHIR endpoints. There are two sub-tabs on this tab: Profile Filtering and Resource Filtering. Both tabs display the same data but provide different filtering options either via profile or resource. Clicking on the FHIR endpoint displays the popup window containing details of the FHIR endpoint.

9. Cap Stat Size Tab

Displays the distribution of the sizes (in bytes) of Capability Statements received from FHIR endpoints per developer.

10. Validations Tab

Displays the distribution of validation results (success or failure) of FHIR endpoints across custom data validation rules defined in Lantern. You can view details of these rules here. This page also displays details regarding the validation failures for the FHIR endpoints failing at a specific validation check.

11. Security Tab

Displays the statistics of FHIR endpoints with regards to different security-related measures. Additionally, you can see the distribution of FHIR endpoints across different security authorization codes. Finally, you can view security details for all FHIR endpoints in the table at the bottom.

12. SMART Response Tab

Displays the statistics of FHIR endpoints with regards to different SMART response-related measures. Additionally, you can see the distribution of FHIR endpoints that provide well-known URL across developers along with another distribution across SMART Response capabilities. Finally, you can view SMART response-related details for all FHIR endpoints in the table at the bottom.

13. Contact Information Tab

Displays the Contact details like email, telephone, etc. for all FHIR endpoints.

14. Downloads / API Tab

Provides the CSV files that are available on the Endpoints and Organizations tabs as well. Additionally, it provides a link to the onc-open-data GitHub repository which contains the daily CSV files from the past months for the Endpoints tab data. Finally, it also provides API endpoints and instructions on filters which can be used to programmatically download Endpoints and Organizations data.
